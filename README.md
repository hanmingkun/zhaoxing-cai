# README
## Project Name
College Student Financial Companion AI Bot — Zhao Cai Miao

## Project Introduction
This project is a lightweight, non-marketing financial companion AI designed for college students. It focuses on living expense management, savings planning, consumption control, low-risk financial literacy, anti-fraud education, and emotional support. It is campus-oriented, advertising-free, and does not recommend high-risk investments.

## Core Features
1. Living expense budgeting and monthly spending review
2. Custom savings goal planning and daily target breakdown
3. Low-risk financial product guidance suitable for students
4. Campus loan and investment fraud prevention education
5. First-time investment risk preference assessment reminder
6. Positive emotional support for financial anxiety

## UI & Tech Specs
- Built with HTML, CSS, JavaScript (pure frontend)
- Mobile-friendly responsive layout
- China Merchants Bank brand colors: red & gray
- AI chat, quick questions, billing, savings plan modules
- No backend or database required

## How to Use
1. Open the HTML file in any browser
2. Deploy to a static hosting service for online access
3. Generate a QR code for the URL to enable mobile scanning
4. Use quick buttons or chat to ask financial questions

## Compliance & Safety Rules
- No stocks, futures, crypto, or high-risk products
- No marketing, account opening guidance, or transfer instructions
- No profit promises or professional investment advisory services
- All content is for educational and planning purposes only

## Target Users
Current undergraduate and graduate students needing help with budgeting, saving, spending control, and beginner-friendly finance guidance.
